To create simple tags.

p {ENTER}

Result:  <p></p>

div {ENTER}

Result: 

To create element with class

p.classname {ENTER}

Result:  <p class="classname"></p>

To create element with ID

p#idName {ENTER}

Result:  <p id="idName"></p>

To create element with ID & Class

p#idName.classname {ENTER}

Result:  <p id="idName" class="classname"></p>

To create element with attribute

p[src="cat.jpg"] {ENTER}

Result:  <p src="cat.jpg"></p>

To create element with content text

p{This is a paragraph} {ENTER}

Result:  <p>This is a paragraph.</p>

To create sibling elements

div+div+div {ENTER}

Result:  <div></div> <div></div> <div></div>

To create children elements

div>div>div {ENTER}

Result: 
 <div>
   <div>
      <div>

      </div>
   </div>
</div>

To create elements with climbing elements

div+div>p>span+em^bq {ENTER}

Result: 
<div></div>
<div>
   <p>
      <span> </span>
      <em> </em>
   </p>
   <blockquote> </blockquote>
</div>

Grouping elements

div>(header>ul>li>a)+footer>p {ENTER}

Result: 
<div>
   <header>
      <ul>
         <li>
            <a>
            </a>
         </li>
      </ul>
   </header>
   <footer>
      <p> </p>
   </footer>
</div>

Adding multiple elements using Multiplication

ul>li*5 {ENTER}

Result: 
<ul>
   <li></li>
   <li></li>
   <li></li>
   <li></li>
   <li></li>
</ul>

Adding multiple elements in sequence

ul>li{Item: $}*5 {ENTER}

Result: 
<ul>
   <li>Item: 1</li>
   <li>Item: 2</li>
   <li>Item: 3</li>
   <li>Item: 4</li>
   <li>Item: 5</li>
</ul>

You can do the same for Id's or Classes

ul>li#item${Item: $}*5 {ENTER}

Result: 
<ul>
   <li id="item1">Item: 1</li>
   <li id="item2">Item: 2</li>
   <li id="item3">Item: 3</li>
   <li id="item4">Item: 4</li>
   <li id="item5">Item: 5</li>
</ul>

You can do the same in REVERSE!

ul>li#item$@-1{Item: $@-1}*5 {ENTER}

Result: 
<ul>
   <li id="item5">Item: 5</li>
   <li id="item4">Item: 4</li>
   <li id="item3">Item: 3</li>
   <li id="item2">Item: 2</li>
   <li id="item1">Item: 1</li>
</ul>

Adding multiple elements in sequence Starting at multiple X

ul>li{Item: $@6}*5 {ENTER}

Result: 
<ul>
   <li>Item: 6</li>
   <li>Item: 7</li>
   <li>Item: 8</li>
   <li>Item: 9</li>
   <li>Item: 10</li>
</ul>

You can do the same for Id's or Classes

ul>li#item$@6{Item: $@6}*5 {ENTER}

Result: 
<ul>
   <li id="item6">Item: 6</li>
   <li id="item7">Item: 7</li>
   <li id="item8">Item: 8</li>
   <li id="item9">Item: 9</li>
   <li id="item10">Item: 10</li>
</ul>
